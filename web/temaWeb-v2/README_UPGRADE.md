# temaWeb-v2 (분리 프로젝트)

이 디렉토리는 기존 `temaWeb`와 **완전히 분리된 업그레이드 작업 공간**입니다.

- 원본 소스: `tmp_04app_repo/temaWeb`
- 작업 대상: `web/temaWeb-v2`
- 기존 `invest-recommand`와는 별개로 개발/배포

다음 단계(업그레이드 예정):
1. API 응답 캐시 + 월/일 단위 조회 최적화
2. 테마 랭킹/추세 분석(최근 n일 상승 빈도, 거래대금 변화)
3. 날짜별 히스토리 캘린더 + 클릭 상세
4. 관심 테마 북마크/알림 훅
5. 백테스트용 스냅샷 저장 포맷 표준화
